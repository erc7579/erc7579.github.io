# getMFAValidatorMockSignature

Get the mock signature for the multi-factor authentication validator in order to use for ERC-4337 gas calculations.

## Usage

```typescript copy
const mockSignature = getMFAValidatorMockSignature()
```

## Parameters

None

## Returns

### mockSignature

- Type: [`Hex`](https://viem.sh/docs/glossary/types#hex)

The mock signature for the multi-factor authentication validator.

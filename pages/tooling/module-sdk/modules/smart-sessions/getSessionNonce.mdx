# getSessionNonce

Get the nonce of a session.

## Usage

```typescript copy
const nonce = await getSessionNonce({
  client,
  account,
  permissionId: '0x1234...',
})
```

## Parameters

### permissionId

- Type: [`Hex`](https://viem.sh/docs/glossary/types#hex)

The permission ID of the session to the nonce for.

### client

- Type: `PublicClient`

The rpc client to use for the chain you want.

### account

- Type: [`Account | Address`](/tooling/module-sdk/glossary/types.mdx#account)

The account object or simply the account address.

## Returns

### nonce

- Type: `Promise<bigint>`

Returns the nonce of the given session.

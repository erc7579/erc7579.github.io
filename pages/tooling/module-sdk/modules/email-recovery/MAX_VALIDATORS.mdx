# MAX_VALIDATORS

Returns the maximum number of validators that can be set for email recovery.

## Usage

```typescript
const maxValidators = MAX_VALIDATORS
```

## Parameters

None

## Returns

### maxValidators

- Type: `number`

The maximum number of validators that can be set to execute transactions.

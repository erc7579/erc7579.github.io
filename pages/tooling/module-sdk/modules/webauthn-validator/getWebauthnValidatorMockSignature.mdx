# getWebauthnValidatorMockSignature

Get the mock signature for the WebAuthn validator in order to use for ERC-4337 gas calculations.

## Usage

```typescript copy
const mockSignature = getWebauthnValidatorMockSignature()
```

## Parameters

None

## Returns

### mockSignature

- Type: [`Hex`](https://viem.sh/docs/glossary/types#hex)

The mock signature for the WebAuthn validator.

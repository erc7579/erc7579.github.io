# getDeadmanSwitchValidatorMockSignature

Get the mock signature for the deadman switch validator in order to use for ERC-4337 gas calculations.

## Usage

```typescript copy
const mockSignature = getDeadmanSwitchValidatorMockSignature()
```

## Parameters

None

## Returns

### mockSignature

- Type: [`Hex`](https://viem.sh/docs/glossary/types#hex)

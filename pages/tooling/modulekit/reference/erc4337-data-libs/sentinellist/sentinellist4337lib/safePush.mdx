---
title: "safePush"
---


Add an item to the list and initialize it if it is not initialized.

## Usage

```solidity
list.safePush(account, address(newEntry));
```

## Parameters

### self

- Type: `SentinelList`

The storage pointer of the list.

### account

- Type: `address`

The account address.

### newEntry

- Type: `address`

The entry to be added to the list.

## Returns

None

title: "push"
---

Add an item to the list.

## Usage

```solidity
list.push(account, address(newEntry));
```

## Parameters

### self

- Type: `SentinelList`

The storage pointer of the list.

### account

- Type: `address`

The account address.

### newEntry

- Type: `address`

The entry to be added to the list.

## Returns

None

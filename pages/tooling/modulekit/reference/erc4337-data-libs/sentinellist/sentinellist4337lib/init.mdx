title: "init"
---


Initialize the linked list. This function must be called before adding items to the list.

## Usage

```solidity
list.init(account);
```

## Parameters

### self

- Type: `SentinelList`

The storage pointer of the list.

### account

- Type: `address`

The account address.

## Returns

None

title: "pop"
---


Remove an item from the list.

## Usage

```solidity
list.pop(SENTINEL, address(popEntry));
```

## Parameters

### self

- Type: `SentinelList`

The storage pointer of the list.

### prevEntry

- Type: `address`

The address of the entry before the entry to be removed.

### entry

- Type: `address`

The entry to be removed.

## Returns

None

title: "push"
---


Add an item to the list.

## Usage

```solidity
list.push(bytes32(keccak256("newEntry")));
```

## Parameters

### self

- Type: `bytes32`

The storage pointer of the list.

### newEntry

- Type: `bytes32`

The entry to be added to the list.

## Returns

None

Deploys an account directly instead of during the first UserOperation.

## Usage

```solidity
instance.deployAccount();
```

## Parameters

### instance

- Type: [`AccountInstance`](../../glossary/types#accountinstance)

The instance of the account used for testing.

## Returns

None

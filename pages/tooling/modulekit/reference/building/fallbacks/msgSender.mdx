title: "_msgSender"
---

Get the address of the caller to the account, for example to implement access control. Inherited from `ERC7579FallbackBase`.

## Usage

```solidity
address originalSender = _msgSender();
```

## Parameters

None

## Returns

### sender

- Type: `address`

The address of the caller to the account.

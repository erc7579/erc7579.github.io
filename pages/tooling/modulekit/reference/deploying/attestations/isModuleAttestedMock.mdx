title: "isModuleAttestedMock"
---

Checks if a module is mock attested to.

## Usage

```solidity
bool isAttested = isModuleAttestedMock({ module: module });
```

## Parameters

### module

- Type: `address`

The address of the module to check.

## Returns

### isAttested

- Type: `bool`

True if the module is attested, false otherwise.

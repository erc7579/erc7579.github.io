title: "setSchemaUID"
---
Set the schema UID for an attestation.

## Usage

```solidity
setSchemaUID({
    _schemaUID: schemaUID
});
```

## Parameters

### \_schemaUID

- Type: `SchemaUID | bytes32`

The schema UID to set.

## Returns

None

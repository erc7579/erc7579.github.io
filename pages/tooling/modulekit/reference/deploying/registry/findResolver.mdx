title: "findResolver"

Get the currently used resolver from the Registry.

## Usage

```solidity
ResolverUID resolverUID = findResolver();
```

## Parameters

None

## Returns

### resolverUID

- Type: `ResolverUID | bytes32`

The resolver UID.

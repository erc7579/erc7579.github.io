title: "setResolverUID"
---

Set the Resolver UID to use for the deployment.

## Usage

```solidity
setResolverUID({
    _resolverUID: resolverUID
});
```

## Parameters

### \_resolverUID

- Type: `ResolverUID | bytes32`

The Resolver UID to set.

## Returns

None

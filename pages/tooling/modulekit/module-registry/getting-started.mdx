title: "Getting Started"
---

The Module Registry is built using Foundry. To use it, clone the [git repository](https://github.com/rhinestonewtf/registry) and follow the steps below.

## Installation

To install dependencies, run:

```bash
pnpm install
```

## Quick Start

To build the project, run:

```bash
pnpm build
```

To run tests, run:

```bash
pnpm test
```
